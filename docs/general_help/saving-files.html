Saving Files

Save your work using the File menu:

  • File > Save (Ctrl+S): Saves the current graph. Acts like 'Save As...' for new graphs.
  • File > Save As... (Ctrl+Shift+S): Saves the graph to a specific file path.

Graphs are saved as XML (.xml), including nodes, connections, values, paths, states, and view settings.

An asterisk (*) in the window title indicates unsaved changes.

(Handled by FileManager and NodeGraph)
