Selecting Nodes

Select nodes to interact with them (move, duplicate, delete, edit properties):

  • Single Click (LMB): Selects a single node. Any previously selected nodes will be deselected.
  • Shift + Click (LMB): Adds a node to the current selection without deselecting others.
  • Ctrl + Click (LMB): Toggles the selection state of a single node (selects if unselected, deselects if selected) without affecting others.
  • Box Selection (LMB Drag): Click and drag on an empty area of the canvas to draw a selection box (green border). Nodes intersecting the box will be selected, deselecting any previously selected nodes.
  • Box Selection Additive (Shift + LMB Drag): Hold Shift while dragging a selection box to add the nodes intersecting the box to the current selection.
  • Box Selection Toggle (Ctrl + LMB Drag): Hold Ctrl while dragging a selection box to toggle the selection state of nodes intersecting the box.
  • Box Deletion (Ctrl + RMB Drag): Hold Ctrl and drag with the Right Mouse Button to draw a red box (red border). All nodes intersecting this box will be deleted when you release the button.
  • Select All (Ctrl+A): Standard keyboard shortcut to select all nodes.

Selected nodes have an orange dashed border highlight.

Clicking a node's header in the Node Editor panel also selects it in the graph view.

(Handled by SelectionManager)
