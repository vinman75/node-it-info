Context Menus

Right-clicking in the Node Graph View provides context-sensitive options based on where the click occurs:

  • Canvas Right-Click (on an empty area):
    • This action opens the node creation menu.
    • Nodes within this menu are organized by their functional category (e.g., "Operators", "Data", "Geometry").
    • Navigate through the submenus and click on a desired node type to create an instance of it. The new node will be placed at the scene position where you right-clicked.
  • Node Item Right-Click (directly on a node):
    • If the node has properties that are designated as "keyable attributes" (meaning they can be animated), an "Animation" submenu will appear.
    • This "Animation" submenu provides options such as:
      • Animate Property: Enables animation tracking for a specific keyable attribute of the node. This typically creates an initial keyframe at the current time on the timeline, using the property's current value.
      • Clear All Keys: Removes all existing keyframes from an already animated property of that node.
      • Remove from Sequencer: Disables animation tracking for the property and removes it from the list in the Keyframe Sequencer panel. This action also clears any existing keyframes for that property.
    • Note: Nodes with keyable attributes display a small "K" indicator. This indicator changes color (e.g., to green) if the node currently has active keyframes on any of its properties.
    • Other node-specific actions might appear here depending on the node's type, but standard operations like Delete, Duplicate, or Mute are generally accessed via keyboard shortcuts or other UI elements (e.g., the Node Editor panel, main menu, or toolbar).

(The canvas context menu for node creation is primarily managed by the ContextMenuManager, which interacts with the NodeGraphView and the node registry. Animation-related context menu actions on nodes are also handled by the ContextMenuManager, which then typically invokes commands that interact with the KeyframeManager and the application's undo stack.)
