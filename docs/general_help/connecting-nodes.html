Connecting Nodes

Nodes process data by passing it through connections (wires) between output and input ports (dots).

  • Create Connection: Click and drag with the Left Mouse Button from an output dot (typically on the right side of a node) towards an input dot.
  • Compatibility Feedback: While dragging, the temporary wire and potential target dots indicate compatibility:
    • Yellow (dotted line): Standard dragging line when not hovering over a dot.
    • Green (solid line): Hovering over a compatible input dot. A colored circle indicator () also appears. Releasing will create the connection.
    • Red (solid line): Hovering over an incompatible input dot (wrong data type or attempting to connect to another output dot). Releasing will not connect.
  • Completing Connection: Release the mouse button while hovering over a compatible input dot (typically on the left side of another node) to create the connection.
  • Cancelling Connection Drag: Release the mouse button over an empty area or press Esc to cancel the drag.
  • Connection Style: Connections are drawn as Bezier curves by default. Toggle between Bezier and Linear via View > Connection Style > Linear/Bezier or Ctrl+B.
  • Data Flow: Data generally flows from left to right, from output ports to input ports. Cycles are prevented.
  • Single Input Connection: Input ports only accept one connection. Making a new connection to an already connected input automatically disconnects the old one first.
  • Muted Connection Style: Connections attached to a muted node (input or output side) are drawn with a dashed line style.
  • Port Tooltips: Hovering over an input or output dot shows a tooltip with the port name and its accepted/output data type(s) (e.g., "Input: float", "Output: str", "Input: list").

(Handled by ConnectionManager)
