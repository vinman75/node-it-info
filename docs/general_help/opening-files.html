Opening Files

Load previously saved graphs using File > Open... (Ctrl+O).

Graphs are saved in an XML format (.xml extension).

Loading a file replaces the current graph. You'll be prompted to save unsaved changes.

The application loads node types, names, positions, connections, data values, file paths, mute states, pinned status, and view settings.

If a node type from the file isn't registered, it may fail to load.

(Handled by FileManager and NodeGraph)
