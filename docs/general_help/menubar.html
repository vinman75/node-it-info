Menu Bar

The menu bar at the top of the application window provides access to most application functions, organized into standard menus:

  • File:
    • New Scene (Ctrl+N): Clears the current graph and starts fresh.
    • Open (Ctrl+O): Loads a graph from an XML file.
    • Save (Ctrl+S): Saves the current graph to its existing file.
    • Save As... (Ctrl+Shift+S): Saves the current graph to a new file.
    • Quit (Ctrl+Q): Exits the application.
  • Edit:
    • Undo (Ctrl+Z): Reverts the last action.
    • Redo (Ctrl+Y): Re-applies the last undone action.
    • Duplicate (Ctrl+D): Duplicates the currently selected node(s).
  • View:
    • Set Canvas Background...: Opens a color dialog to change the background color of the Visualizer canvas.
    • Show/Hide Grid (X key): Toggles the visibility of the background grid in the Node Graph View and Visualizer Canvas.
    • Snap to Grid (Ctrl+X): Toggles whether nodes snap to the grid when moved.
    • Connection Style (Sub-menu):
      • Linear: Sets connection lines to be straight.
      • Bezier: Sets connection lines to be curved (default). (Shortcut: Ctrl+B toggles this setting)
    • Toggle Rounded Corners (Ctrl+R): Toggles between rounded and square corners for nodes.
    • Reset Layout: Restores the default arrangement of dockable panels and the Visualizer's canvas background color.
  • Windows:
    • Show/Hide Node Editor: Toggles the visibility of the Node Editor panel.
    • Show/Hide Output Window: Toggles the visibility of the Output Window panel.
    • Show/Hide Visualizer: Toggles the visibility of the Visualizer panel.
    • Show/Hide Keyframe Sequencer: Toggles the visibility of the Keyframe Sequencer panel.
    • Dock Node Editor (Sub-menu):
      • Dock Left: Docks the Node Editor to the left side.
      • Dock Right: Docks the Node Editor to the right side.
  • Execution:
    • Run Graph Execution: Evaluates the entire graph and then executes any final 'action' nodes (e.g., file exports). Also available on the Toolbar.
  • Help:
    • Documentation (F1): Opens this help documentation.
    • About: Displays information about the application.
