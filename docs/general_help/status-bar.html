Status Bar

The Status Bar at the bottom of the main window provides quick information:

  • Total Nodes: Current count of nodes in the graph.
  • Selected Nodes: Number of currently selected nodes.
  • Temporary Messages: Shows brief status updates (e.g., "Executing graph...", "Export successful...", errors).

(Managed by StatusBarManager)
