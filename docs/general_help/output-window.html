Output Window

The Output Window panel (docked at the bottom by default) displays information from Logger and ChatLogger nodes.

  • Logger/ChatLogger Tabs: Each specific Logger or ChatLogger node creates a dedicated tab, named after the node's current name.
  • Messages (Logger): Logger nodes send their input value (as string) to their tab when evaluated or when the Update button is clicked.
  • Messages (ChatLogger): ChatLogger nodes display paired user prompts and AI responses when the AI response is received.
  • Tab Controls: Each tab has buttons on the left:
    • Update (Logger only): Forces the Logger node to log its current value again.
    • Clear: Clears the text in the current tab.
  • Closing Tabs: Use the 'x' button. Closed tabs are recreated if the corresponding node logs again or Update is clicked (for Logger).
  • Automatic Renaming: Tabs update their name if the corresponding Logger/ChatLogger node is renamed.

Use Logger nodes for general value inspection and ChatLogger specifically for AI interaction pairs.

(Managed by OutputWindow class)
