AI Graph Builder

The AI Graph Builder allows you to generate graph structures from natural language descriptions using a configured AI model (like Google Gemini).

  • Location: A dedicated toolbar usually appears at the bottom of the main window.
  • Usage:
    1. Enter a description of the desired graph into the input field (e.g., "Load text from file_path.txt, convert it to uppercase, and log the result").
    2. Click the Build button ().
    3. The application sends your request along with information about available nodes to the AI.
    4. If successful, the AI returns a description of nodes and connections, which the application then uses to automatically create and arrange the graph.
  • Node Manifest: The AI uses a manifest of available nodes (types, descriptions, inputs, outputs) provided by the application to select appropriate nodes.
  • Connection Hints: For some nodes, specific connection instructions ('hints') are provided to the AI, which it MUST follow for correct graph construction (e.g., how to connect ChatLogger).
  • Requirements/Errors: This feature requires a properly configured AI API key (e.g., `GEMINI_API_KEY` environment variable). Errors from the AI or during graph construction will be shown via messages or the status bar.

This provides a quick way to prototype or generate common graph patterns.

(Managed by AIGraphBuilderManager using GeminiWorker)
