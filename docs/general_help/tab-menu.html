Tab Menu Search

The Tab Menu provides a fast, searchable way to add nodes:

  • Activation: Press the Tab key when the graph view is focused (and not inside an input widget like a text field). The menu appears near the mouse cursor.
  • Filtering: Start typing to filter the list of available nodes. The filter matches against both the node type name (e.g., "Add", "FloatToInt") and its category (e.g., "Operators", "Conversion").
  • Navigation: Use the Up/Down Arrow Keys or PageUp/PageDown to navigate the filtered list.
  • Tooltips: Hovering over a node type in the list displays its descriptive tooltip.
  • Creation: Press Enter or Double-Click the desired node type to create it at the mouse cursor's current scene position.
  • Creation with Connection: If you activate the Tab Menu while dragging a connection wire from an output dot, selecting a node will attempt to automatically connect the dragged wire to the first compatible input on the newly created node.
  • Closing: Press Esc, click outside the menu, or move the mouse cursor away from the menu area to close it without creating a node. Pressing Tab again also closes it.

(Functionality provided by TabMenu and TabEventFilter)
