Add Nodes to the Graph

There are two primary ways for users to add new nodes to the graph:

  1. Context Menu (Right-Click): Right-clicking on an empty area of the graph canvas brings up a menu. Nodes are organized by category (e.g., 'Operators', 'Data', 'Conversion'). Navigate the submenus and click on a node type (e.g., 'Add', 'Float', 'FloatToInt') to create an instance of it at the location you clicked.
  2. Tab Menu Search (Tab Key): Pressing the Tab key while the graph view is focused opens a searchable menu near your mouse cursor. Start typing to filter the list of available nodes, then press Enter or double-click to create the selected node type. See the 'Tab Menu Search' section for more details.

You can also use the AI Graph Builder (see Interface Panels section) to generate multiple nodes and connections from a text description.

Newly created nodes typically receive default names (e.g., "Float1", "Add2"), which can be changed in the Node Editor panel.

(Functionality provided by ContextMenuManager, TabMenu, AIGraphBuilderManager interacting with NodeGraphView)
