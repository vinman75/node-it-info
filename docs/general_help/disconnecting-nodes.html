Disconnecting Nodes / Cut Tool

You can remove connections between nodes in several ways:

  1. Drag Off Input: Click and drag an existing connection wire *away* from the input dot it's connected to using the Left Mouse Button. A temporary red dotted line will appear. Release the mouse button over an empty area of the canvas to complete the disconnection.
  2. Reconnecting Input: Making a new connection to an input port that already has a connection will automatically disconnect the previous one.
  3. Cut Tool: Hold Ctrl + Shift and drag with the Left Mouse Button across one or more connection wires. A red dashed line will appear. Releasing the mouse button will disconnect all connection wires intersected by this cut line.
  4. Deleting Node: Deleting a node automatically removes all connections attached to it.

(Handled by ConnectionManager and SelectionManager)
