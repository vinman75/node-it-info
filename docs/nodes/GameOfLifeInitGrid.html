GameOfLifeInitGrid

Category: Simualtion

Description: Generates a 2D grid to initialize a Game of Life simulation. The grid's 'Width', 'Height', randomization 'Seed', and initial live cell 'Density' (0.0-1.0) can be set using attributes in the Node Editor. Each of these internal settings can be overridden by connecting a value to the corresponding 'Width In', 'Height In', 'Seed In', or 'Density In' input port. If a port is connected, its value takes precedence over the internal Node Editor setting for that parameter. Outputs a 'ValueGrid' (NumPy integer array where 1 typically means alive, 0 dead).

Inputs:

  • Width In: int
  • Height In: int
  • Seed In: int
  • Density In: float

Outputs:

  • Output: ValueGrid
