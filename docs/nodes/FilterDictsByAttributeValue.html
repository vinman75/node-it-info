FilterDictsByAttributeValue

Category: List

Description: Takes 'List In'. The 'Attribute Key' to filter by can be selected from a dynamic dropdown or overridden by 'Key Override' input. Compares this attribute's value against 'Filter Value' using the selected 'Filter Mode'. Outputs a new list containing only matching dictionaries. Returns empty list on error.

Inputs:

  • List In: list
  • Key Override: str
  • Filter Value: any

Outputs:

  • Output: list
