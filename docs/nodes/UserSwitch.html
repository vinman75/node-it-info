UserSwitch

Category: Flow_Control

Description: Outputs data from one of its 10 input ports ('Input 0' to 'Input 9'). The active input is selected globally using number hotkeys: '1' selects 'Input 0', '2' selects 'Input 1', ..., '9' selects 'Input 8', and '0' selects 'Input 9'. The current active state (1-10) is displayed on the node.

Inputs:

  • Input 0: any
  • Input 1: any
  • Input 2: any
  • Input 3: any
  • Input 4: any
  • Input 5: any
  • Input 6: any
  • Input 7: any
  • Input 8: any
  • Input 9: any

Outputs:

  • Output: any
