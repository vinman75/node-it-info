Timer

Category: Animation

Description: A resettable timer driven by 'Delta Time'. Outputs elapsed time, normalized progress (0-1), a boolean indicating if it's running, and a single-frame 'Finished Pulse' when the 'Duration' is reached. Can be set to loop. Internal Duration/Loop can be overridden by inputs.

Inputs:

  • Delta Time: float
  • Start: bool
  • Reset: bool
  • Duration Override: float
  • Loop Override: bool

Outputs:

  • Output: dict

Extractable Attributes (from output data):

  • elapsed
  • normalized
  • is_running
  • finished_pulse
