WeavePattern

Category: Geometry

Description: Creates paths representing warp and weft threads in a plain weave, defined by Width, Height, Num Warp/Weft. 'Gap Fraction' controls overlap space. The pattern is generated locally (bottom-left at 0,0) and then positioned/rotated/scaled by the 'Transform' input. Outputs a list of 2-point paths.

Inputs:

  • Width: float
  • Height: float
  • Num Warp: int
  • Num Weft: int
  • Gap Fraction: float
  • Transform: dict

Outputs:

  • Output: list
