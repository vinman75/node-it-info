DrawPath

Category: Drawing

Description: Takes path data ('Path Data') and draws lines connecting the points. Input can be a single path (list of [x,y]), a list of paths (list of lists of [x,y]), OR structured data (List[Dict{'path': List[[x,y]], ...}]). Optionally takes 'Color', 'Opacity', and 'Thickness' (single value or list matching paths). Outputs draw commands.

Inputs:

  • Path Data: list
  • Color: any
  • Opacity: any
  • Thickness: any

Outputs:

  • Output: list
