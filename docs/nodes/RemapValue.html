RemapValue

Category: Utility

Description: Takes a 'Value (0-1)', 'Dest Min', and 'Dest Max'. Outputs the value linearly mapped to the [Dest Min, Dest Max] range. Value input is clamped 0-1.

Inputs:

  • Value (0-1): float
  • Dest Min: float
  • Dest Max: float

Outputs:

  • Output: float
