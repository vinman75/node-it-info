AppendToList

Category: List

Description: Takes a list ('List In') and an item ('Item'). Creates and outputs a *new* list with the item added to the end. Returns an empty list if 'List In' is not a list.

Inputs:

  • List In: list
  • Item: any

Outputs:

  • Output: list
