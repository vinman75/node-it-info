ExtractImageChannel

Category: Compositing

Description: Takes an 'Image In' (premultiplied float32 RGBA NumPy array) and outputs the selected channel as a 'FloatGrid' (2D NumPy array HxW, float32, 0-1 range). For R,G,B, values are un-premultiplied first. Channel selected via dropdown in Node Editor.

Inputs:

  • Image In: Image

Outputs:

  • Output: floatgrid
