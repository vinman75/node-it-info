ReactionDiffusionStep

Category: Simualtion

Description: Takes current chemical concentration grids (as a dictionary {'grid_a':..., 'grid_b':...} via 'Grids In'), simulation parameters ('Diffusion A/B', 'Feed Rate', 'Kill Rate', 'Delta Time'), and a 'Frame In' trigger. Outputs the next state of the chemical grids in the same dictionary format. Handles 'Reset In' to re-initialize from 'Initial Grids In' (also a dictionary) or internal defaults.

Inputs:

  • Grids In: dict
  • Frame In: int
  • Reset In: bool
  • Initial Grids In: dict
  • Diffusion A: float
  • Diffusion B: float
  • Feed Rate: float
  • Kill Rate: float
  • Delta Time: float

Outputs:

  • Output: dict
