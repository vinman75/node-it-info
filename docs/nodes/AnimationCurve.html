AnimationCurve

Category: Animation

Description: Takes a 'Time' input (typically 0-1, will be clamped) and outputs an eased 'Value' (0-1) based on the 'Curve Type' selected in the Node Editor. Use 'RemapValue' or 'FitRange' on the output to scale to desired animation range.

Inputs:

  • Time: float
  • Curve Type Override: str

Outputs:

  • Output: float
