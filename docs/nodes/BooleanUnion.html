BooleanUnion

Category: Geometry

Description: Performs a boolean union: merges all input shapes into a single geometric entity. Inputs 'Shape In 0', 'Shape In 1', etc., should be closed polygons (List[[x,y]]) or lists of such polygons. Dynamic inputs are added as needed. Outputs the resulting geometry as a list of polygons. Requires 'shapely' library.

Inputs:

  • Shape In 0: list
  • Shape In 1: list

Outputs:

  • Output: list
