CombineXYZ

Category: Vector3

Description: Takes three numeric inputs (X, Y, Z) and combines them into a list representing a Vector3: [x, y, z]. Invalid or disconnected inputs default to 0.0.

Inputs:

  • X: float
  • Y: float
  • Z: float

Outputs:

  • Output: vector3
