TruchetTile

Category: Generator

Description: Creates a grid of Truchet tiles. 'Grid X/Y', 'Tile Size', 'Seed', and 'Line Width' can be set in the Node Editor or overridden by input ports. 'Output Mode' selects between raw paths or thicker stroke polygons. The pattern can be globally transformed using the 'Transform' input.

Inputs:

  • Grid X In: int
  • Grid Y In: int
  • Tile Size In: float
  • Seed In: int
  • Line Width In: float
  • Transform In: dict

Outputs:

  • Output: list
