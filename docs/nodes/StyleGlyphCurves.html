StyleGlyphCurves

Category: Text

Description: Takes 'Styled Glyph Curve Data In' (from TextLayoutNode) and applies global and/or per-glyph style overrides for fill/stroke color, opacity, stroke width, local scale, and local rotation. Per-glyph overrides use an 'Index Mode' to map values from input lists. Outputs the modified styled glyph curve data.

Inputs:

  • Styled Glyph Curve Data In: list
  • Global Fill Color: any
  • Global Stroke Color: any
  • Global Stroke Width: float
  • Global Opacity: float
  • Global Scale: any
  • Global Rotation: float
  • Character Fill Colors: list
  • Character Stroke Colors: list
  • Character Stroke Widths: list
  • Character Opacities: list
  • Character Scales: list
  • Character Rotations: list

Outputs:

  • Output: list

Extractable Attributes (from output data):

  • char
  • char_index_in_string
  • word_index_in_string
  • line_index
  • baseline_start
  • advance_width
  • bounding_box_local
  • curve_data_local
  • fill_color_rgba
  • stroke_color_rgba
  • stroke_width
  • scale_local
  • rotation_degrees_local
