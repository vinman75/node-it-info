FitRange

Category: Utility

Description: Linearly maps the input 'Value' from the range defined by ['Source Min', 'Source Max'] to the range ['Dest Min', 'Dest Max'].

Inputs:

  • Value: float
  • Source Min: float
  • Source Max: float
  • Dest Min: float
  • Dest Max: float

Outputs:

  • Output: float
