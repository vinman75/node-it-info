DrawPixelGrid

Category: Drawing

Description: Takes 'Color Grid' data. If 'Image' type (NumPy RGBA array), a 'Channel View' dropdown allows displaying R,G,B,A, or Luminance as grayscale. If 'FloatGrid' or 'ValueGrid' (integer grid), displays as grayscale. If Python list (legacy List[List[color_values]]), 'Pixel Size' applies. 'Position' sets top-left. Outputs a single draw command.

Inputs:

  • Color Grid: ['Image', 'floatgrid', 'ValueGrid', 'list', 'any']
  • Position: vector2
  • Pixel Size: float

Outputs:

  • Output: list
