VonNeumannGalaxyEvolver

Category: Simualtion

Description: Manages galaxy grid and probes. Steps simulation when 'FrameTriggerIn' changes. Handles emission, movement, colonization, termination. Parameters set in Node Editor, or can be overridden by corresponding input ports.

Inputs:

  • FrameTriggerIn: int
  • ResetSim: bool
  • InitialSystemGridIn: ValueGrid
  • EmissionIntervalIn: int
  • ProbeSpeedIn: float
  • ProbeMaxAgeIn: int
  • MaxTotalProbesIn: int
  • ProbeReplicationCountIn: int
  • ProbeDetectionRadiusIn: float

Outputs:

  • Output: dict
