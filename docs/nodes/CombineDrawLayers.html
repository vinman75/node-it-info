CombineDrawLayers

Category: Drawing

Description: Merges inputs. If an input is an 'Image' (NumPy array), it's wrapped as a draw command. If it's a list, it's assumed to be drawing commands. Layer 0 is drawn first.

Inputs:

  • Layer 0: Image
  • Layer 1: Image

Outputs:

  • Output: list
