Switch

Category: Flow_Control

Description: Outputs the value from the 'Input X' port corresponding to the 'Index'. E.g., Index=0 outputs 'Input 0'. Starts with a minimum number of 'Input X' ports; more are added automatically as you connect to the last available one (up to ~5). If Index is out of range or invalid, outputs 'Default'.

Inputs:

  • Index: int
  • Default: any
  • Input 0: any
  • Input 1: any

Outputs:

  • Output: any
