MotionBlurCompositor

Category: Drawing

Description: Evaluates the 'Draw Data In' branch multiple times within a shutter interval defined by 'Shutter Open/Close' and 'Sub Steps'. Requires 'Current Frame' input. Composites results by layering semi-transparent versions from each sub-step. Select this node to view the result. Works best with Additive Blending enabled in Visualizer.

Inputs:

  • Draw Data In: ['list', 'dict', 'any']
  • Current Frame: int
  • Shutter Open: float
  • Shutter Close: float
  • Sub Steps: int

Outputs:

  • Output: list
