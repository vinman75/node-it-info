InstanceOnPoints

Category: Geometry

Description: Takes 'Prototype Geometry' (single path OR list of paths) and places a transformed copy at each point in 'Instance Points'. Uses optional 'Instance Rotations' (degrees) and 'Instance Scales' (uniform factor) lists to modify each instance. Outputs a list of all resulting polygons. Rotation and scaling are applied relative to the prototype's origin (0,0).

Inputs:

  • Instance Points: list
  • Prototype Geometry: list
  • Instance Rotations: list
  • Instance Scales: list

Outputs:

  • Output: list
