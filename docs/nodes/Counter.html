Counter

Category: Animation

Description: Maintains an internal integer count. 'Increment'/'Decrement' inputs trigger changes by 'Step Value'. 'Reset' input sets count to 'Initial Value'. Optional 'Min Value' and 'Max Value' clamp the count. Internal defaults can be overridden by corresponding input ports.

Inputs:

  • Increment: bool
  • Decrement: bool
  • Reset: bool
  • Initial Value Override: int
  • Step Value Override: int
  • Min Value Override: int
  • Max Value Override: int

Outputs:

  • Output: int
