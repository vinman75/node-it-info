JitterPoints

Category: Geometry

Description: Takes 'Points In' (single path, list of paths, or structured data), a 'Seed', and max offset amounts ('Amount X', 'Amount Y'). Outputs a new list where each point has a random offset added within the specified amount range. Preserves input structure (including attributes). If 'Points In' is a list of dictionaries, attributes are preserved and can be accessed using 'ExtractListAttributeNode'.

Inputs:

  • Points In: list
  • Seed: int
  • Amount X: float
  • Amount Y: float

Outputs:

  • Output: list
