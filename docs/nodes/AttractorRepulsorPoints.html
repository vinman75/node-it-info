AttractorRepulsorPoints

Category: Transformation

Description: Takes 'Points In' (single path, list of paths, or structured data), 'Attractor Points', 'Strength', 'Radius', and 'Falloff'. Outputs geometry where each point is moved towards (+Strength) or away (-Strength) from the nearest attractor within the radius. Preserves input structure (including attributes). If 'Points In' is a list of dictionaries, attributes are preserved and can be accessed using 'ExtractListAttributeNode'.

Inputs:

  • Points In: list
  • Attractor Points: list
  • Strength: float
  • Radius: float
  • Falloff: float

Outputs:

  • Output: list
