VonNeumannGalaxySetup

Category: Simualtion

Description: Creates the initial 'System Grid' for the Von Neumann simulation. Places 'Sol' (initial colonized system) at 'Sol Position'. Scatters 'Num Random Systems' at pseudo-random locations based on 'Systems Seed'. Grid dimensions determined by 'Width' and 'Height'. All parameters are internal but can be overridden by input ports.

Inputs:

  • Width: int
  • Height: int
  • Num Random Systems: int
  • Systems Seed: int
  • Sol Position: vector2

Outputs:

  • Output: ValueGrid
