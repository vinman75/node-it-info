FlipNormals

Category: Geometry

Description: Takes 'Paths In' (single path, list of paths, or structured data with 'path' keys). Outputs new geometry where the order of points in each path is reversed. This is useful for correcting or inverting the normal direction of 2D polygons, which is often determined by winding order.

Inputs:

  • Paths In: list

Outputs:

  • Output: list
