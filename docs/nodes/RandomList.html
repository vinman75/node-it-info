RandomList

Category: Generator

Description: Creates a list containing 'Count' number of pseudo-random floats. Each float uses the provided integer 'Seed' and falls within the range ['Min', 'Max']. Defaults: Count=10, Seed=0, Min=0.0, Max=1.0.

Inputs:

  • Count: int
  • Seed: int
  • Min: float
  • Max: float

Outputs:

  • Output: list
