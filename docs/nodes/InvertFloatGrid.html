InvertFloatGrid

Category: Compositing

Description: Takes a 'FloatGrid In' (2D float32 NumPy array) and inverts its values using the formula (1.0 - value), clipping the result to the 0-1 range. Outputs the inverted 'FloatGrid'.

Inputs:

  • FloatGrid In: floatgrid

Outputs:

  • Output: floatgrid
