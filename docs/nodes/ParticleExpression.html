ParticleExpression

Category: Dynamics

Description: Iterates through 'Particle List In'. For each particle (dict), executes the 'Expression' string, allowing modification of attributes like 'pos', 'vel', 'age', 'life', 'mass', 'size', 'color', 'opacity', 'rotation', etc. Outputs the modified 'Particle List Out'. Use 'ExtractListAttributeNode' to get lists of specific particle attributes.

Inputs:

  • Particle List In: list
  • Expression: str
  • A: any
  • B: any
  • C: any
  • D: any

Outputs:

  • Output: list

Extractable Attributes (from output data):

  • id
  • pos
  • vel
  • age
  • life
  • max_life
  • mass
  • inv_mass
  • size
  • radius
  • color
  • opacity
  • rotation
  • _base_color
  • _base_opacity
  • _base_size
  • _color_ramp_data
  • _opacity_ramp_data
  • _size_ramp_data
