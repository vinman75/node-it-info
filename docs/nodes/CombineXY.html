CombineXY

Category: Vector2

Description: Takes two numeric inputs (X, Y) and combines them into a list representing a Vector2: [x, y]. Invalid or disconnected inputs default to 0.0.

Inputs:

  • X: float
  • Y: float

Outputs:

  • Output: vector2
