Merge

Category: Compositing

Description: Combines 'Image A' and 'Image B' using the chosen 'Operation'. The 'Mix Factor' (0-1) blends between 'Image A' (at 0) and the full operation result (at 1). An optional 'Mask In' (alpha/grayscale) spatially controls the mix. All inputs are processed as premultiplied float32 RGBA. Handles shape mismatches. Outputs premultiplied float32 RGBA NumPy array.

Inputs:

  • Image A: Image
  • Image B: Image
  • Mask In: Image
  • Mix In: float

Outputs:

  • Output: Image
