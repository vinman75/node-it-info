RenderStyledGlyphs

Category: Drawing

Description: Takes 'Styled Glyph Curve Data In' (from StyleGlyphCurvesNode) and converts it into a list of 'polygons' draw commands for the CanvasWidget. Supports 'Fill', 'Outline', or 'Fill and Outline' draw modes. Applies an optional 'Global Transform' to the entire text block.

Inputs:

  • Styled Glyph Curve Data In: list
  • Global Transform: dict

Outputs:

  • Output: list
