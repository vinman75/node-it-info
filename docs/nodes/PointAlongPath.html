PointAlongPath

Category: Geometry

Description: Takes input path data ('Path Data') and a factor (0-1). Accepts single paths (List[[x,y]]), lists of paths (List[List[[x,y]]]), or structured data (List[Dict{'path':...}]). Uses the *first* path found. Outputs the [x, y] coordinates of the point at that fractional distance along the path's total length. Factor 0=start, 1=end.

Inputs:

  • Path Data: list
  • Factor: float

Outputs:

  • Output: vector2
