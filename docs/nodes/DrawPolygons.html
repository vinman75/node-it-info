DrawPolygons

Category: Drawing

Description: Takes polygon data ('Polygons') and draws them. Input can be a single polygon (list of [x,y]), a list of polygons (list of lists of [x,y]), OR structured data (List[Dict{'path': List[[x,y]], ...}]). Optionally takes 'Fill Color'/'Opacity', 'Stroke Color'/'Opacity', and 'Thickness'. Outputs draw commands.

Inputs:

  • Polygons: list
  • Fill Color: any
  • Fill Opacity: any
  • Stroke Color: any
  • Stroke Opacity: any
  • Thickness: any

Outputs:

  • Output: list
