FitRangeList

Category: Utility

Description: Linearly maps each numeric value in 'Value List' from ['Source Min', 'Source Max'] to ['Dest Min', 'Dest Max']. Non-numeric items passed through.

Inputs:

  • Value List: list
  • Source Min: float
  • Source Max: float
  • Dest Min: float
  • Dest Max: float

Outputs:

  • Output: list
