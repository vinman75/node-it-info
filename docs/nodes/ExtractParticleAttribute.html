ExtractParticleAttribute

Category: Dynamics

Description: Specifically for dynamic data like particle data from a solver. Takes a list of dictionaries/objects ('Particle List In') or a NumPy particle data dictionary. The 'Attribute' to extract is selected from a dropdown (populated from data keys) or overridden by 'Attribute Override' input. This node ALWAYS re-evaluates its input to ensure fresh data from dynamic sources.

Inputs:

  • Particle List In: list
  • Attribute Override: str

Outputs:

  • Output: list
