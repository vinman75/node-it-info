ParticleEmitter

Category: Dynamics

Description: Emits particles. Emission mode, geo normal usage, spread, and initial global attributes (speed, life, size, mass, color, opacity, rotation) are set via Node Editor, with optional override input ports. Requires Delta Time and connection TO a Physics Solver.

Inputs:

  • Reset Emitter: bool
  • Position: vector2
  • Source Geometry: list
  • Emission Direction: vector2
  • Rate: float
  • Count: int
  • Spread Angle Override: float
  • Speed Override: float
  • Lifespan Override: float
  • Size Override: float
  • Mass Override: float
  • Color Override: vector3
  • Opacity Override: float
  • Rotation Override: float
  • Ramp Color: list
  • Ramp Size: list
  • Ramp Opacity: list
  • PP Speed: list
  • PP Lifespan: list
  • PP Size: list
  • PP Mass: list
  • PP Color: list
  • PP Opacity: list
  • PP Initial Rotation: list

Outputs:

  • Output: emitter_ref

Extractable Attributes (from output data):

  • id
  • pos
  • vel
  • age
  • life
  • max_life
  • mass
  • inv_mass
  • size
  • radius
  • color
  • opacity
  • rotation
