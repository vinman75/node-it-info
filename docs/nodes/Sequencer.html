Sequencer

Category: Animation

Description: Steps through a 'Sequence Values' list. 'Advance Mode' (Node Editor) determines if it steps each evaluation or on 'Step Trigger' (rising edge). 'End Behavior' (Node Editor) controls what happens at sequence end. 'Reset' input restarts the sequence. Internal sequence can be set in Node Editor or overridden by 'Sequence Override' input.

Inputs:

  • Sequence Override: list
  • Step Trigger: bool
  • Reset: bool
  • Default Output: any

Outputs:

  • Output: any
