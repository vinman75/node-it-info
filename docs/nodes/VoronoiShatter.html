VoronoiShatter

Category: Geometry

Description: Takes input shape(s) (single polygon or list of polygons) and 'Seed Points'. Generates a Voronoi diagram, clips it to the combined shape bounds (+padding), and outputs a list of polygons representing the fragments of the input shape(s) intersected with each Voronoi cell. Requires 'scipy' and highly recommends 'shapely' library.

Inputs:

  • Shape In: list
  • Seed Points: list
  • Padding: float
  • Num Random Points: int
  • Random Seed: int

Outputs:

  • Output: list
