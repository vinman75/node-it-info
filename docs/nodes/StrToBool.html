StrToBool

Category: Conversion

Description: The StrToBool node takes a string input and outputs a boolean. Recognizes 'true', '1', 'yes', 'on' as True, and 'false', '0', 'no', 'off' as False (case-insensitive). If the input is invalid or None, it outputs False.

Inputs:

  • Input: str

Outputs:

  • Output: bool
