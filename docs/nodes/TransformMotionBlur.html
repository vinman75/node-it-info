TransformMotionBlur

Category: Drawing

Description: Takes 'Prototype Geometry' and an 'Animated Transform In' dictionary. For blur (Sub Steps > 1), it samples the transform at multiple sub-steps within the shutter interval and generates thin, elongated polygons (streaks) connecting vertex positions between sub-frames. The 'Streak Color' input defines streak color. The 'Fill Color' input defines the fill for the non-blurred (Sub Steps = 1) version. If Sub Steps = 1 (no blur), it draws the prototype once with the 'Fill Color'. Outputs a composite of these draw commands. Best with Additive Blending in Visualizer for blur.

Inputs:

  • Prototype Geometry In: list
  • Animated Transform In: dict
  • Current Frame: int
  • Shutter Open: float
  • Shutter Close: float
  • Sub Steps: int
  • Streak Color: vector3
  • Fill Color: vector3

Outputs:

  • Output: list
