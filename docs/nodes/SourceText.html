Category: Text

Description: Outputs text. Uses direct input or 'Text In' port. Reveal controlled by 'Reveal Mode' (None, By Character, By Word). If 'Count/Progress Mode' is 'Progress', 'Reveal Value' (0-1 float) drives animation. If 'Count', 'Reveal Value' (integer) directly sets items to show.

Inputs:

  • Text In: str
  • Reveal Value: any

Outputs:

  • Output: str
