VectorMultiply

Category: Vector3

Description: Takes two Vector3 inputs ('Vec A', 'Vec B') and outputs a new Vector3 where each component is the product of the corresponding input components. Invalid inputs default to [0, 0, 0].

Inputs:

  • Vec A: vector3
  • Vec B: vector3

Outputs:

  • Output: vector3
