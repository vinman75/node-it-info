TranslatePoints

Category: Transformation

Description: Takes input geometry ('Points In') and translates it by an offset. The offset is determined primarily by the 'Transform In' dictionary input (using its 'translate' key). If 'Transform In' is disconnected or invalid, the 'Translation' Vector2 input is used instead. Preserves input structure. If 'Points In' is a list of dictionaries, attributes are preserved and can be accessed using 'ExtractListAttributeNode'.

Inputs:

  • Points In: list
  • Translation: vector2
  • Transform In: dict

Outputs:

  • Output: list
