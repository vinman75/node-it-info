ExtractGlyphPaths

Category: Text

Description: Takes 'Styled Glyph Curve Data In'. For each glyph, it reconstructs its defining curve paths, applies local scale/rotation and baseline positioning, then an optional 'Global Transform'. Outputs a flat list of these final paths (polygons made of line segments). This data includes both outer boundaries and inner hole boundaries, suitable for DrawPathNode.

Inputs:

  • Styled Glyph Curve Data In: list
  • Global Transform: dict

Outputs:

  • Output: list
