StylePathsByAttribute

Category: Drawing

Description: Takes structured path data (List[Dict{'path':..., 'attr':val, 'parent_attr':pval}]). Normalizes 'Attribute Key' (for color) and 'Parent Attribute Key' (for start thickness) using 'Map Min/Max'. Samples 'Color Ramp' (using 'Attribute Key') and 'Thickness Ramp' (using BOTH keys). Applies taper along 'Subdivisions' based on start/end thickness and 'Taper Factor'. Outputs styled 2-point path draw commands. Use 'FilterDictsByAttributeValue' or 'ExtractListAttributeNode' to access attributes from input data if needed.

Inputs:

  • Structured Paths In: list
  • Attribute Key: str
  • Parent Attribute Key: str
  • Map Min: float
  • Map Max: float
  • Thickness Ramp: list
  • Thickness Scale: float
  • Color Ramp: list
  • Color Opacity: any
  • Subdivisions: int
  • Taper Factor: float

Outputs:

  • Output: list

Extractable Attributes (from output data):

  • type
  • data
  • color
  • thickness
  • fill_color
  • stroke_color
