ExportImageSequence

Category: File

Description: Renders 'Draw/Image Data' input per frame, matching the Visualizer's Resolution Gate area. Uses Resolution/Frame Range from the Visualizer Dock by default. Connect 'Override' inputs to use specific settings. Path pattern set in Node Editor. Triggered by 'Run Graph' or Export button.

Inputs:

  • Draw/Image Data: ['list']
  • Override Width: int
  • Override Height: int
  • Override Start Frame: int
  • Override End Frame: int

Outputs:

  • None
