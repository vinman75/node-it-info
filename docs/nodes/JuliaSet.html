JuliaSet

Category: Generator

Description: Calculates a Julia set for a given constant 'k' (K_Real, K_Imag) and view (Center, Scale) at a specified Resolution and Iteration depth. Outputs a grid of integer escape-time values ('ValueGrid').

Inputs:

  • K_Real: float
  • K_Imag: float
  • CenterX: float
  • CenterY: float
  • Scale: float
  • ImageWidth: int
  • ImageHeight: int
  • MaxIterations: int

Outputs:

  • Output: ValueGrid
