Vector2Subtract

Category: Vector2

Description: Takes two Vector2 inputs ('Vec A', 'Vec B') and outputs a new Vector2 where each component is the result of subtracting B's component from A's (A - B). Invalid inputs default to Vector2(0, 0).

Inputs:

  • Vec A: vector2
  • Vec B: vector2

Outputs:

  • Output: vector2
