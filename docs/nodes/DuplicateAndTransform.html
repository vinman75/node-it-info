DuplicateAndTransform

Category: Geometry

Description: Takes 'Geometry In' (single path or list of paths), duplicates it 'Num Copies' times. Each copy is transformed relative to the *previous copy* by 'Translate', 'Rotate' (degrees), and 'Scale', using 'Center' as the origin. Outputs a flat list containing all original and transformed paths.

Inputs:

  • Geometry In: list
  • Num Copies: int
  • Translate: vector2
  • Rotate: float
  • Scale: float
  • Center: vector2

Outputs:

  • Output: list
